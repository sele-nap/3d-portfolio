 # 3D Portfolio

Personal portfolio built with React, TypeScript and Three.js.

## Stack
- React
- TypeScript
- Three.js (@react-three/fiber)

## Status
Work in progress.

## Git workflow
- `main`: stable and deployable version
- `dev`: active development

## Commit convention
This project follows a lightweight conventional commit style:

- `feat:` new feature
- `fix:` bug fix
- `refactor:` code refactoring (no functional change)
- `style:` UI / styling changes
- `chore:` tooling, configuration, dependencies
- `docs:` documentation
- `wip:` work in progress (temporary)
